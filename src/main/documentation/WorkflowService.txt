Workflow - General meaning of workflow is a process in which business processing
           steps will be converted to software execution phases to automate business steps.
           Example : Steps  involved in  receiving order from online to delivery of product  to the customers.

          DSL languages are often used to write instructions. Yaml is such language used in actiegrid.

In active grid application, Workflow term is used in a different context  that scheduling a configuration managment script.
There are two types of confiugration managment plugins are included in existing activegrid application

1. Ansible
2. Puppet

It have been observed that most implementation binded to Ansible, Though general design of the application support both
ansible and puppet plugins but unlike ansible,no  exclusive steps for  Puppet  execution haven't seen in the code.
Compared to ansibl,more puppet related scripts are written and included.

Mcollective an ansible module,included in code base. It enbale parllel execution of tasks and dynamic host descovery over
large distributed network of nodes.

Puppet  support in-mechine or remote system configuration. In active application remote system configuration used.

There is a tab in activegrid user interface to upload module. As of now Ansible and Puppet modules are supported in screen.
